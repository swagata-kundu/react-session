# React-Session demo setup

### This setup is only for development purpose
## Features

* React 16
* Webpack 3
* Babel
* React Router V4
* Hot Module Replacement

## Installation

* `git clone https://github.com/swagata-kundu/react-session.git`
* cd react-session
* npm install
* npm start
* visit `http://localhost:8080/`
